Решить 10 примеров (в тестах) на Stream API, задачки ниже, они немного усложняются сверху вниз

1) Вывести все четные числа в диапазоне от 1 до 100,
2) Умножить каждое число в массиве [1, 2, 3, 4, 5] на 2,
3) Посчитать сумму чисел в массиве [1, 2, 3, 4, 5], используя reduce()
4) Вывести числа в диапазоне от 1 до 50 с шагом 2,
5) Найти первый четный элемент в списке [1, 2, 3, 4, 5],
6) Отсортировать элементы массива [1, 3, 5, 7, 9] по возрастанию,
7) Вывести первые 10 чисел в списке [0, 1, 2, … 99]
8) Пропустить первые 10 элементов списка [0, 1, 2,.., 99] и начать выводить с 11-го элемента, выводя каждый 10-й элемент
9) Выведите на экран все числа в диапазоне от 1 до 100, которые делятся на 3.
10) Выведите все нечетные числа в заданном массиве.
11) Выведите на экран только те элементы списка, которые больше заданного числа.
12) Отфильтруйте все элементы списка, которые меньше 0.
13) Выведите на экран строки, которые начинаются с заданной подстроки.
14) Выведите все числа в списке, которые не кратны 2.
15) Отфильтруйте объекты по определенному свойству, например, выведите все записи из базы данных, у которых значение поля равно 1.
16) Выведите только те элементы коллекции, которые удовлетворяют определенному условию, например, являются уникальными или имеют определенный формат.
17) Отфильтруйте элементы массива, которые не являются числами.
18) Выведите на экран элементы списка, которые не входят в другой список.
19) Проверить, все ли числа в массиве [0, 1, …, 19] являются четными,
20) Проверить, есть ли четное число в списке [0, 1, 2, 3],
21) Собрать все уникальные элементы Stream в список и отсортировать их.
22) Собрать элементы Stream в карту, где ключом будет первая буква каждого слова, а значением — само слово. Отсортировать ключи в алфавитном порядке.
23) Собрать числа в Stream в список сумм цифр каждого числа.
24) Собрать даты в Stream в список, где каждый элемент — это количество дат в каждом месяце года.
25) Собрать файлы в Stream в список, где каждый элемент — количество файлов с определенным расширением.
26) Собрать пользователей в Stream в список объектов, где каждый объект содержит информацию о городе проживания пользователя и количестве пользователей из этого города.
27) Собрать продукты в Stream в таблицу, где каждая строка — это информация о продукте (название, категория, стоимость), отсортированную по категории продукта.
28) Создайте Stream из массива чисел, выведите на экран числа, кратные 3 и 5 одновременно.
29) Создайте два Stream-а: один из массива чисел 1…5, второй из массива 5…10. Объедините эти два Stream-а в один и выведите на экран.
30) Создайте Stream-у чисел от 1 до 20. Создайте новый Stream, который будет выводить на экран только четные числа и числа, кратные 3. Затем объедините эти два Stream-a в один и выведите результирующий Stream.
31) Создайте Stream чисел от 2 до 10. Умножьте их на 2 и выведите результат на экран, ограничьтесь первыми десятью результатами.
32) Создайте три Stream-а из массивов чисел 1…10, 5…20 и 10…30 соответственно. Объедините их в один Stream и выведите числа, которые кратны 3 или 5.
33) Создайте Stream, который выводит на экран четные числа от 2 до 30.
34) Создать стрим из массива чисел и вывести на экран только числа, которые больше 10 и меньше 20.
35) Создать два стрима: один из чисел от 0 до 10, другой из чисел от 10 до 20. Объединить их в один стрим и вывести на экран числа больше 10.
36) Создать стрим чисел от 0 до 100. Умножить их на 2 и вывести на экран результат, ограничиться первыми 10 результатами.
37) Создать стрим, который выводит числа от 1 до 10. Найти среднее арифметическое этих чисел и вывести его на экран.
38) Создать три стрима из массивов чисел от 1 до 10, от 10 до 20 и от 20 до 30 соответственно. Объединить их в один стрим и вывести числа, которые кратны 5.
39) Создать стрим четных чисел от 2 до 40 и вывести на экран количество элементов в этом стриме.
40) Разделить элементы Stream на две группы: четные и нечетные, вывести результаты.
41) Разделить слова в Stream на две группы по первой букве: гласные и согласные, посчитать количество слов в каждой группе.
42) Разделить числа в Stream на три группы по остатку от деления на 3 (0, 1, 2), посчитать сумму чисел в каждой группе.
43) Разделить даты в Stream на четыре группы по временам года, посчитать количество событий в каждом времени года.
44) Разделить файлы в Stream на два списка: с расширением .txt и с расширением .doc, посчитать количество файлов в каждом списке.
45) Разделить пользователей в Stream на группы по городу (ближний и дальний план), посчитать количество пользователей в каждой группе.
46) Разделить продукты в Stream на несколько групп по категориям (например, молочные продукты, мясные продукты, овощи, фрукты), посчитать стоимость продуктов в каждой группе.
47) Создание Stream из массива целых чисел и вывод на экран всех простых чисел в диапазоне от 2 до 100.
48) Создание двух Stream из массивов целых чисел и объединение их в один, затем вывод на экран суммы квадратов элементов этого Stream.
49) Создание трех Stream из массивов и объединение их в один. Затем вывод на экран среднего геометрического значения элементов этого Stream.
50) Создание Stream символов и поиск всех строк, состоящих из строчных букв и цифр.
51) Создание Stream объектов и поиск всех объектов, у которых определенное поле равно определенному значению.
52) Создание Stream дат и поиск всех дат, которые попадают в определенный диапазон.
53) Создание Stream целых чисел и поиск всех чисел, у которых сумма цифр равна заданному числу.
54) Создание Stream строк и поиск всех строк, у которых длина равна заданному числу и которые состоят из определенного набора символов.
55) Создание Stream пар чисел и поиск всех пар, у которых произведение равно заданному числу или которые удовлетворяют другому условию.
56) Создание Stream коллекций и поиск всех коллекций, которые содержат определенный элемент или удовлетворяют другому условию.
57) Сгруппировать элементы Stream по их чётности, посчитать размер каждой группы и вывести результаты.
58) Сгруппировать слова в Stream по первой букве, посчитать количество слов в каждой группе и вывести результаты в виде словаря, где ключ — первая буква слова, а значение — количество слов, начинающихся на эту букву.
59) Сгруппировать числа в Stream по остатку от деления на 3, посчитать сумму чисел в каждой группе и вывести результаты.
60) Сгруппировать даты в Stream по месяцу, посчитать количество дат в каждом месяце и вывести результаты в виде календаря с количеством событий в каждом месяце.
61) Сгруппировать файлы в Stream по расширению, посчитать количество файлов с каждым расширением и вывести результаты в виде списка, где ключ — расширение файла, а значение — количество файлов с таким расширением.
62) Сгруппировать пользователей в Stream по городу проживания, посчитать количество пользователей из каждого города и вывести результаты в виде карты мира с количеством пользователей из каждого города.
63) Сгруппировать продукты в Stream по категории, посчитать стоимость продуктов в каждой категории и вывести результаты в виде таблицы, где столбец — категория продукта, а строка — стоимость продуктов в этой категории.
64) Для любого набора случайно-сгенерированных чисел нужно определить количество парных. Для решения задачи использовать средства программного интерфейса Stream API.
